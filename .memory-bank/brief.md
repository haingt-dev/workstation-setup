# Workstation Setup

## One-Liner
Reproducible automated config for personalized terminal + dev environment on Nobara 42 / Fedora. One-command bootstrap via `setup.sh`.

## Key Facts
| Field | Value |
|-------|-------|
| Target OS | Nobara / Fedora-based only |
| Shell stack | Zsh + Starship + Atuin + tmux (TPM) |
| Terminal | Kitty + Catppuccin themes |
| Tools | zoxide, eza, bat, fzf, ripgrep, fd, lazygit, yazi, Godot, Qdrant, Podman |
| Fonts | Nerd Fonts + JetBrains Mono (user font dir) |
| Idempotency | Re-runnable without breaking existing installs |
| Status | maintained |
