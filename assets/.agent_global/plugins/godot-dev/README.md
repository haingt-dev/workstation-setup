# godot-dev

Godot 4.x development skills for Claude Code.

## Skills

- **godot-gdscript-patterns**: State machines, autoloads, resources, object pooling, component system, scene management, save system, performance tips
- **godot-debugging**: Error interpretation, common bugs, debugging techniques, Godot 4 migration issues

## Install

```bash
claude plugin marketplace add ~/agent
claude plugin install godot-dev@agent --scope project
```

Note: Wildtide's Stop hook (gdformat) is project-specific — kept in Wildtide's `.claude/settings.json`.
